Videdit
"""
Just scrolling through tiktok or instagram you come across a video that is well edited and is cool. You want to edit some of your videos like that but it's to time consuming.
You'll have to cut clips, remove backgrounds from certain images, add transitions - just thinking about the whole process makes you give up entirely and choose a simple edit sequence.
"""
Videdit is designed to aid you, Its a auto-manual editing system that collaborates with you to create those edits that you desire.
It's created for partnership rather than automation.You just input the video with the edited style you desire then, it geneartes a template for you; A template that is editable & customizable. After that you can select the clips you want to add or images that you want to apply to the template. 
The system  partners withyou by giving suggestions and adjustable thresholds.
It's a partnership as you have control over what the system does, you edit, validate and create.
 Design Principles:
  1. Transparency - The system is able to be clear about each process, frame analysis, cut detection and the whole process of template generation.
  2. Human-in-the-loop


This is created using:
- Python
- PySide6 - GUI
- OpenCv
